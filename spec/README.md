# Specifications

The `spec` directory contains the official technical specifications of the Verso framework.

Every architectural component must be specified before it is implemented.

Specifications describe **what a component is**, **its responsibilities**, **its guarantees**, and **how it interacts with the rest of the framework**.

Implementation always comes after specification.
