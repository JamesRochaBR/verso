# Vision

## Why Verso Exists

Large Language Models have fundamentally changed how software is built.

However, the way we provide context to coding agents has changed very little.

Most solutions rely on increasingly larger prompts, duplicated instructions and manually curated context.

As projects grow, this approach becomes expensive, difficult to maintain and inconsistent.

Verso exists to change that.

---

## Mission

Build a modular framework that allows AI coding agents to load only the knowledge they actually need.

---

## Vision

Become the standard architecture for modular AI context management.

---

## Principles

- Context should be intentional.
- Knowledge should be modular.
- Specifications come before implementations.
- Every component should have a single responsibility.
- The framework should remain agent-agnostic.

---

## Long-term Goal

Enable any AI coding agent to understand a software project through structured, reusable and composable knowledge rather than monolithic prompts.
